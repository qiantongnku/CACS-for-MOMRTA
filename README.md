# CACS-for-MOMRTA
This directory contains the main code of the proposed algorithm in the following paper:

T. Qian, X. -F. Liu and Y. Fang, "A Cooperative Ant Colony System for Multiobjective Multirobot Task Allocation With Precedence Constraints," in IEEE Transactions on Evolutionary Computation, vol. 29, no. 3, pp. 734-748, June 2025, doi: 10.1109/TEVC.2024.3364493.

This code is only for academic use and if you use this code, please cite the following paper:
